# webpack-pug-template
Template to quick start a webpack pug template

## Installation
`npm install`
## Dev
`npm run-script build`

Will run a host at 
`localhost:8080`

Now you can edit

`src/index.pug`

`src/css/style.css`

With hot reloading.

## Production
`npm run-script build:prod`

Will make the static html at:

`/dist`
